Data Scraping

Index constituents were downloaded from KRX website and parsed as Pandas Dataframe. Changes to to the indices were also taken into error handling. Historical adjusted prices for KOSPI200, KOSDAQ150 stocks were scraped from Yahoo Finance and Naver Finance, respectively, using Selenium and BeautifulSoup. In order to speed up scraping process, multiprocessing was used and execution time sped up 240 times faster. Alternative sources of data - KRX and Bloomgberg - attempted.

Data Vizualization

Visualization was inspired by static data viz from Financial Times. D3.js library was used. Tooltip to display more data and force layout to prevent bubble overlaps.
